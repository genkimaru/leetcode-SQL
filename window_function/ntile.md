## ntile N等分，给每个等分记录一个编号

*  三等分
```sql
mysql> select id ,num, ntile(3) over (order by id ) as ntile_number from test;
+------+------+--------------+
| id   | num  | ntile_number |
+------+------+--------------+
|    1 |    1 |            1 |
|    2 |    1 |            1 |
|    3 |    1 |            1 |
|    4 |    2 |            2 |
|    5 |    1 |            2 |
|    6 |    2 |            2 |
|    7 |    2 |            3 |
|    8 |    3 |            3 |
|    9 |    3 |            3 |
+------+------+--------------+

```
